Salesforce Loyalty Management Overview

Section Description
Importance of Loyalty Programs
  • Customer Retention: Incentivizes repeat purchases.
  • Brand Advocacy: Turns customers into promoters.
Common Pain Points
  • Separate Systems: Difficulty integrating multiple solutions.
  • Program Tracking: Inaccurate customer lifetime value tracking.
  • Personalization: Lack of tailored experiences.
  • Audience Targeting: Difficulty identifying target audiences for promotions.
  • Timeliness: Struggles in engaging customers at the right moment.
  • Points Management: Complexities in managing points due to high transaction volumes.
Salesforce Loyalty Management Solutions
  • Consolidation: Unified platform for program management.
  • Automation: Streamlines processes to enhance efficiency.
Key Features
  • Member Management: Enrollment and tracking of members.
  • Program Setup: Customization with tiers, currencies, and partners.
  • Loyalty Experiences: Automatic benefit assignment and targeted promotions.
Required Permissions
  • Loyalty Management Permission Set: Grants edit access to relevant objects.
  • Additional permissions configured via profiles or sets.


Set Up a Loyalty Program

Section Details
Loyalty Program Setup
    Set up a loyalty program in the Loyalty Management app You can create and activate the program, which supports both B2C(customers) and B2B (partners) models.
Tier Groups and Tiers
    Create a tier group and associate it with the loyalty program. Define three key attributes: Tier Period, Qualifying Period, and Tier Model (Fixed or Anniversary). Create tiers based on points, offering different rewards per tier.
Program Currencies
    Define Qualifying Points (used for tier progression) and Non-Qualifying Points (redeemable for rewards). Set up expiration rules, like theFixed Model or Activity Model, for non-qualifying points.
Escrow Points
    Use escrow to temporarily hold points after a transaction before crediting them to members, ensuring proper handling in case of returns.
Partners and Partner Products
    Add partners and associate their products with the program. Partners can be accrual partners (members earn points) or redemption partners (members spend points). Products can be assigned individually or by category.
Program Members
    Enroll members by inviting them to sign up. Restrict membership to individuals or allow group membership, where members can contribute points to a shared balance.
Next Step
    Once the loyalty program, tier groups, currencies, partners, and members are set up, the next step is to focus on defining member rewards.
